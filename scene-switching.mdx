## Scene Switching Overview

Scene switching allows you to manage multiple conversation contexts simultaneously, enabling seamless transitions between different topics, projects, or interaction styles with your AI avatar.

## Creating and Managing Scenes

<Steps>
  <Step title="Create New Scene">
    Click the "+" button in the scene selector to create a new conversation context
  </Step>
  <Step title="Name Your Scene">
    Give your scene a descriptive name (e.g., "Marketing Strategy", "Code Review")
  </Step>
  <Step title="Configure Settings">
    Set up avatar behavior, tone, and specific parameters for this scene
  </Step>
  <Step title="Switch Between Scenes">
    Use the scene selector to instantly switch contexts
  </Step>
</Steps>

## Scene Switching Commands

### Quick Commands
- **"Switch to [scene name]"** - Direct scene switching
- **"Create marketing scene"** - Create and switch to new scene
- **"List my scenes"** - Show available scenes
- **"Delete this scene"** - Remove current scene

### Visual Controls
- **Scene Selector Dropdown**: Quick access to all scenes
- **Scene Tabs**: Browser-like tabs for active scenes
- **Scene Cards**: Visual representation of scene contexts
- **Context Indicators**: Visual cues showing current scene

## Best Practices

<Steps>
  <Step title="Organize Logically">
    Group related topics and projects into coherent scenes
  </Step>
  <Step title="Use Descriptive Names">
    Clear scene names help with quick identification and switching
  </Step>
  <Step title="Configure Appropriately">
    Set up avatar behavior that matches the scene's purpose
  </Step>
  <Step title="Regular Maintenance">
    Archive or delete scenes that are no longer active
  </Step>
</Steps>

### Scene Organization Tips
- **Project-Based**: One scene per major project or initiative
- **Role-Based**: Different scenes for different professional roles
- **Topic-Based**: Separate scenes for distinct subject areas
- **Client-Based**: Individual scenes for different clients or stakeholders

## Troubleshooting

<AccordionGroup>
  <Accordion title="Scene Memory Issues">
    - Verify scene isolation settings
    - Check for context bleed between scenes
    - Reset scene memory if needed
    - Review scene configuration
  </Accordion>

  <Accordion title="Switching Problems">
    - Refresh the interface
    - Check browser compatibility
    - Verify scene permissions
    - Clear cached scene data
  </Accordion>

  <Accordion title="Performance Issues">
    - Limit number of active scenes
    - Archive unused scenes
    - Optimize scene configurations
    - Check system resources
  </Accordion>
</AccordionGroup>
