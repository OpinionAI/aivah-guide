# Share Settings Configuration

Share settings allow you to manage how your agents are distributed and accessed by users. This includes managing multiple shared agents, configuring access permissions, and organizing your published agents.

## Shared Agents Management

![Shared Avatar Agents Overview](/Users/ahmedz/Documents/GitHub/chatbot/aivah-guide/share/Share%20/Screenshot%202025-08-09%20at%201.24.15%20PM.png)

The shared agents interface provides a comprehensive view of all your published agents:

### Agent Gallery View
- **Visual Overview**: See thumbnails and details of all shared agents
- **Distribution Types**: Each agent shows its sharing method (Custom, Chat Bubble, iframe, Direct Link)
- **Agent Information**: View agent name, avatar, and LLM model for each agent
- **Pagination**: Handle large collections with organized page navigation (12 agents per page)

### Agent Organization Features
- **Rows per page**: Customize how many agents display per page (12 default)
- **Navigation Controls**: Move between pages of agents efficiently
- **Agent Count**: Track total number of shared agents (e.g., "0 - 0 of 0")

3. **Customized URLs**: Links with specific parameters and configurations

## Agent Configuration Management

### Profile Information
For each shared agent, you can manage:
- **Agent Name**: Display name (e.g., "Product Presenter")
- **Avatar Name**: Character identity (e.g., "Sales1")
- **LLM Model**: AI model selection (e.g., "GPT-4o Mini Realtime")
- **Distribution Method**: How the agent is shared

## Publishing Status Management

### Publication States
- **Draft**: Agents in development, not yet published
- **Published**: Live agents available to users
- **Paused**: Temporarily disabled agents
- **Archived**: Retired agents kept for reference

### Access Control
- **Public Access**: Open to anyone with the link
- **Restricted Access**: Limited to specific users or groups
- **Private Testing**: Internal testing before public release
- **Scheduled Publishing**: Automated publishing at specific times
